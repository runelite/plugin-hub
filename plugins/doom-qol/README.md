# Doom of Mokhaiotl Quality of Life
### Overview:
#### Visualize Volatile Earth and orb paths to aid positioning and decision making
#### Visualize fallen debris tiles
#### Helps outline debris [rocks] placed in the arena.

### Features:
#### "Mark Debris on Ground [Toggle]"
> (On by default)
* Should the [tiles of the] fallen debris in the arena be marked.
#### "Debris Color"
* Color selection for user customization.
#### "Fill Debris Tiles [Toggle]"
> (On by default)
* Whether the *hull* of the tile is filled in, as opposed to just the outline.
#### "Debris fill opacity"
* Optional opacity of the hull, value between 0 (outline only) to 255 (filled), utilizes the same color as the outline.
#### "Mark Shadow Tiles [Toggle]"
>(On by default)
* The falling debris once the boulder shatters is highlighted on the floor.
#### "Shadow Color"
* Color selection for user customization (default: red to easily distinguish and avoid).
#### "Show Shield Path [Toggle]"
> (On by default)
* Whether the given Earthen Shield safety path will be highlighted - Center-tile of shield (1x1) by default
#### "Shield Path Color"
* Color selection for user customization (default: green to easily distinguish safety).
#### "Show full 3x3 Orb Path [Toggle]"
> (Off by default)
* Whether the full 3x3 outline of the Earthen Shield's safety path will be visually highlighted
