\# Shop Unlocks

Locks in-game shop items until they are naturally obtained through gameplay, including looting, crafting, questing, or finding in the world.

Shop Unlocks is intended to add a little more difficulty to the Ironman early game by eliminating the convenience of shops and encouraging exploration and unique grinds.



\## Features



* Locks all store items by default until they have been obtained through gameplay, automatically unlocking when they enter the player's inventory.
* Allows manual unlocking and locking via \*\*Shift + Right-Click\*\*.
* Highlights locked shop items with a configurable overlay.
* Provides option to reset all unlocks if desired.



\## Config Options

* \*\*Show Overlay\*\* - Used to turn locked item overlay on or off.
* \*\*Locked Item Color\*\* - Adjust the color of the locked item overlay using a color picker.
* \*\*Overlay Opacity\*\* - Adjust transparency of the overlay.
* \*\*Reset All Unlocks\*\* - Resets all unlocks if the user types "yes" and then selects yes from prompt.
