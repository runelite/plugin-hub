# MyIP's Idle Crab

A RuneLite plugin that tracks sand crab aggro time and aggressively alerts you when you go out of combat.

## Features
- 10 minute sand crab aggro timer
- Color-coded countdown (green → yellow → red)
- Progress bar
- Strong out-of-combat alarm
- Auto reset when leaving crab tiles
- Optional aggro timer persistence across world hops

## Configuration
All behavior is configurable via the RuneLite plugin settings.

## Compliance
This plugin is informational only.  
It does not automate gameplay or interact with game input.
